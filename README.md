 🧮 Calculadora de IMC

Este projeto é uma calculadora de IMC (Índice de Massa Corporal) desenvolvida com o objetivo de praticar e aprimorar habilidades em JavaScript, HTML e CSS.


 
 🚀 Funcionalidades

* Inserir nome, altura e peso
* Calcular automaticamente o IMC
* Exibir o resultado com base nos dados informados
* Feedback visual simples e direto para o usuário





 💻 Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript





 🎯 Objetivo do projeto

Este projeto foi criado com o foco em:

* Praticar manipulação do DOM
* Trabalhar com entrada de dados do usuário
* Aplicar lógica de programação em JavaScript
* Melhorar a organização de código em projetos simples




 📂 Estrutura do projeto

* `index.html` → estrutura da página
* `style.css` → estilização
* `script.js` → lógica da aplicação




 
 ✨ Melhorias futuras
